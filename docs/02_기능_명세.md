# 기능 명세

## 1. 인증/보안
- 팀원별 개인 계정 로그인 (ID/PW, 8자 이상)
- 관리자/일반 사용자 권한 분리
- 세션 24시간 유지
- 모든 행동 감사 로그 기록 (data/audit.log)

## 2. 법인 관리
- 관리자가 54개 법인 등록 및 담당자 지정
- 일반 사용자는 본인 담당 법인만 접근 가능

## 3. 엑셀 업로드 & 스마트 파서
- 법인마다 다른 엑셀 양식을 **자동 인식**
  - 헤더 행 자동 탐지 (1~60행 범위 탐색)
  - 컬럼 자동 매핑: 거래처/수취인/거래처명 → payee, 금액/원화금액 → amount 등
  - 합계/소계 행 자동 제거
- 여러 파일 동시 업로드 (Ctrl+클릭)
- 파일 유형 구분:
  - **과거 데이터**: 패턴 학습용 (누적, 덮어쓰기 아님)
  - **신규 거래**: 오늘 승인 건 분석용

## 4. 이상거래 감지 규칙

| 규칙 | 가중치 |
|------|--------|
| 5억원 이상 초고액 | +3 |
| 1억원 이상 고액 | +2 |
| 5천만원 이상 | +1 |
| 법인 평균 대비 5배 초과 | +2 |
| 상위 5% 금액의 2배 초과 | +2 |
| 신규 거래처 (이력 없음) | +1 |
| 신규 거래처 + 1천만원 이상 | +1 |
| 동일 수취인 당일 3건 이상 | +1 |
| 동일 금액 3건 이상 반복 | +1 |
| 적요에 긴급/urgent 포함 | +1 |
| 계좌번호 미기재 + 100만원 이상 | +1 |

**판정 기준**: 4점 이상 = 이상 / 2~3점 = 주의 / 0~1점 = 정상

## 5. 분석 결과 화면
- 이상/주의/정상 건수 및 총금액 카드
- 법인 지급 패턴 카드 (건당 평균, 일평균, 월평균, 최대 단건, 학습 건수)
- 이상·주의 거래 목록 (감지 사유 태그)
- 수취인별 요약
- 위험도 필터 (이상/주의/정상)
- 수취인/계좌번호 검색
- CSV 내보내기
- 분석 이력: 법인별 최근 30개 결과 서버 저장
