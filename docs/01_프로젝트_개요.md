# 프로젝트 개요

## 목적
지주사 재무팀 6명이 54개 그룹사 법인의 자금 최종승인 전 이상거래를 감지하는 웹 시스템(FDS).

## 운영 방식
- 각 담당자가 약 10개 법인씩 담당
- 법인에서 엑셀로 올라오는 지출결의서를 업로드하여 분석
- 과거 데이터로 패턴 학습 → 신규 거래 건에 대해 이상 여부 판정

## 기술 스택
- **Backend**: Python Flask + openpyxl + pandas + numpy
- **Frontend**: HTML/CSS/JS (순수 JS, 프레임워크 없음)
- **데이터**: 파일 기반 JSON 저장 (DB 미사용)
